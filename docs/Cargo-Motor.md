# 货运马达 {#cargo-motor}

货运马达是[货运管理](/Cargo-Management)分类中的物品。

它是货运系统相关物品的合成中间件。

## 获取 {#obtaining}

货运马达可以在[增强型工作台](/Enhanced-Crafting-Table)中一次性制作4个。

## 使用 {#usage}

货运马达可以用在以下配方中:

* [货运管理器](/Cargo-Manager)
* [货运节点(连接器)](/Connector-Node)
* [高级货运节点(输出)](/Advanced-Output-Node)
* [反应堆访问接口](/Reactor-Access-Port)
* [垃圾箱](/Trash-Can)
* [流体泵](/FLuid-Pump)
